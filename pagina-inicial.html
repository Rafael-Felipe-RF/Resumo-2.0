Criando o código base

Para começar, ao abrir o Visual Studio Code coloque um ponto de ! (exclamação) e aperte Enter, vai aparecer o seguinte código.

<DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">        
<meta name="viewport" content
="width=device-widthinitial-scale=1.0">
<title>Document</title>
</head>
<body>

Conteúdo do Site!

</body>
</html>
  • Devemos mudar o parâmetro Html Lang para <html lang="pt-br">.
  • Em <title> devemos colocar o título da nossa página, que será exibido na aba do navegador.
  • Todo o conteúdo da página deverá ser colocado entre a tag <body></body>.
  • Para colocar uma linha usa <hr>
  • Para quebrar a linha <br>

Símbolos e Emoji

  • Para o símbolo de abre e fecha tag (<>) ser exibido na página, usa-se o código & l t ; (sem espaço) para abrir e & g t ; (sem espaço) para fechar.
  • Para adicionar um comentário usa < !-- comentário -- > o que estiver escrito entre os traços não será exibido na página. ou use o atalho: + +
  • Para o símbolo de Marca Registrada usa (& reg) ®
  • Para Copyright (& copy) ©
  • Para Trademark (& trade) ™
  • Para Delta usa (& Delta iniciando em maiúsculo) Δ
  • Para acesso à mais símbolos acesse o Misc symbols

Favicon

O Favicon, é um ícone que é exibido na aba do navegador, antes do título da página, deve sempre ser colocado dentro da Tag <head>

  • Para criar o seu Favicon você vai no site favicon.io, nesse site você pode criar seu ícone de favorito e baixar, sempre em formato .ico.
  • Para ter acesso a biblioteca de ícones visite o site icon archive.
  • Para desenhar seu ícone vai em favicon.cc .
  • Após criar o ícone crie a tag link:Favicon: 
    <link rel="shortcut icon" 
href="favicon.ico" 
type="image/x-icon">
    O Favicon deve estar salvo na sua pasta.

Formatação

  • Para usar Itálico use <em>
  • Para usar Negrito use <strong>
  • Para texto inserido ou sublinhado use <ins>
  • Para texto Excluído use <del>
  • Para usar Marca Texto use <mark>

    • Para mudar a cor use <mark style="background-color: Cor;"></mark>

  • Para texto Sobescrito use <sup>

    • Ex: x2+b5+c3=0

  • Para texto Subescrito use <sub>

    • Ex: Fórmula química do ácido Sulfúrico: H2SO4

  • Para exibir código Fonte use <code> com a tag <pre> dentro:

    • <code>
    <pre>Código</pre>
    </code>

  • Quando indicar um endereço use <adress>

    • EX: Rua das Acácias 225 - Belo Horizonte - MG

  • Para Abreviações use <abbr>
  • Para citações simples use <q>

    • Ex: Só sei que nada sei.

Listas

  • Para cada item de uma lista use <li>

listas Ordenadas

Use a tag <ol> Para criar uma lista ordenada: 

<ol>
<li>Arroz</li>
<li>Café</li>
<li>Açúcar</li>
<li>Pão</li>
</ol>

O resultado deve ser exibido assim:

  1. Arroz
  2. Café
  3. Açucar
  4. Pão

Podemos mudar o tipo do índice com a tag <type>.

Os tipos aceitos são 1 A a I i

  • 1 para números
  • A (maiúsculo) para letras maiúsculas.
  • a (minúsculo) para letras minúsculas.
  • I (maiúsculo) para números romanos maiúsculos.
  • i (minúsculo) para números romanos maiúsculos.

Listas Não Ordenadas

Para listas não ordenadas use <ul>

<ul>
<li>Arroz</li> 
<li>Café</li> 
<li>Açúcar</li> 
<li>Pão</li> 
</ul>

O resultado deve ser exibido assim:

  • Arroz
  • Café
  • Açúcar
  • Pão

Também podemos mudar o tipo do índice com a tag <type>.

Os tipos aceitos são:

  • Disc para um disco preenchido
  • Circle para um círculo não preenchido
  • Square para um quadrado

Lista De Definição

  • Para criar a lista de definição use a tag <dl>
  • Para criar o item use a tag <dt>
  • Para criar a descrição use a tag <dd>

Exemplo:

<dl>
<dt>Arroz</dt>>
<dd>Água, Arroz, óleo, alho, sal</dd>>
<dt>Feijão</dt>>
<dd>Água, Feijão, Óleo, Sal</dd>>
</dl>

O resultado deve ser exibido assim:

Arroz
Água, Arroz, óleo, alho, sal
Feijão
Água, Feijão, Óleo, Sal

Escondendo uma lista

Podemos esconder uma lista com a tag <details>.

Para definir o que será exibido usamos a tag <summary>.

Exemplo de código:

<details>
<summary>Clique aqui para ver a lista
</summary>
<ul>
<li>Pão</li>
<li>Café</li>
<li>Arroz</li>
<li>Macarrão</li>
</ul>
</details>

O resultado deve ser exibido assim:

Clique aqui para ver a lista
  • Pão
  • Café
  • Arroz
  • Macarrão

Imagens

No CSS, o whidth (tamanho) da imagem, deve sempre ter 100%

  • Sempre faça 3 tamanhos da imagem e salve na pasta imagens, isso serve para diferentes tamanhos de telas.
  • A tag de imagem é <picture>
  • Dentro da tag <picture> usa a tag <img>
  • Para definir os tamanhos de imagem, use <source:media:type> listando os tamanhos das imagens da menor para a maior.
                        
<picture>

<source media="(max-whidth: 750px)" 
srcset= "local do arquivo" 
type="tipo da imagem">

<source media="(max-whidth: 1050px)" 
srcset= "local do arquivo" 
type="tipo da imagem">
<img src= "local da imagem" 
alt= "texto alternativo">

<picture>
  • Sempre modifique o termo em destaque, ele vem por padrão com MIN, coloque Max.
  • Sempre atente a ordem do tamanho das imagens, na tag <img> fica o tamanho original da imagem, que deve ficar na base, depois cololque os tamanhos menores.

Tabelas

  • Para definir uma tabela use a tag <table>.
  • Para definir um Cabecalho para a tabela use a tag <thead>, para definir as linhas do cabeçalho, use a tag <th>.
  • Para definir o Corpo da tabela use a tag <tbody>.
  • Para definir o rodapé da tabela use a tag <tfoot>.
  • Cada linha da tabela é definida pela tag <tr> (Table Row).
  • cada coluna é definida pela tag <td>(Dado da Tabela).
  • Para definir uma borda para a tabela use a definição na tag, <table>, border. É uma definição obsoleta, o ideal é fazer a configuração pelo CSS.

Exemplo de Código:

Passe o mouse nos pontilhados para explicação;

<table border="1">
    <thead>
        <th>Filme</th>
        <th>Ano de Lançamento</th>
        <th>Avaliação</th>
    </thead>

    <tbody>
        <tr>
            <td>Homem de Ferro</td>
            <td>2008</td>
            <td>8,9</td>
        </tr>

        <tr>
            <td>Thor</td>
            <td>2010</td>
            <td>8,0</td>
        </tr>

        <tr>
            <td>Vingadores</td>
            <td>2012</td>
            <td>10</td>
        </tr>

    <tfoot>
        <td colspan="3">Filmes da Marvel
        </td>

    < /tfoot >
    </tbody >
</table >

O resultado deve ser exibido assim:

Filme Ano de Lançamento Avaliação
Homem de Ferro 2008 8,9
Thor 2010 8,0
Vingadores 2012 10
Filmes da Marvel

Para definir um efeito zebrado na tabela, utiliza uma Pseudo classe.

tbody tr:nth-child(2n){
background-color:  Defina uma cor;
}

Para intervalos de 2 em 2, ao invéz de usar um número, use ODD, (Par) ou EVEN, (Ímpar)

Para fixar o cabeçalho da tabela no topo da página, serve para tabelas grandes.

A tabela tem que ter nas CSS's a position: relative;

thead > tr > th{
position: sticky;
top: 0;
background-color: "defina uma cor"
(a mesma do Thead);
}

Para configurar uma coluna especifica usa a tag do HTML <colgroup>

                        
        <colgroup>
            <col class="nome">
            <col class="sexo">
            <col class="idade">
            <col class="profissao">
        </colgroup>

Para fazer a tabela ficar mais responsiva usa a definição overflow-x

        
No Html, envelopa a tabela em uma Caixa (div ou section)

        <div id="container">
            <table>...
            </table>
        </div>
        
        
Nas CSS's coloque a definição:

        #container{
        overflow-x: auto;
        }
